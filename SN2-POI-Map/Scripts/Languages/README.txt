SN2 POI Map language files

Canonical translation files:
- en.lua       English fallback
- de.lua       German
- es-419.lua   Latin American Spanish
- fr.lua       French
- it.lua       Italian
- ja.lua       Japanese
- ko.lua       Korean
- pt-BR.lua    Brazilian Portuguese
- ru.lua       Russian
- uk.lua       Ukrainian
- zh.lua       Simplified Chinese

Compatibility aliases:
- Spanish variants such as es.lua, es-LA.lua, es-MX.lua, and other Latin American regional tags load es-419.lua.
- Portuguese alias pt.lua loads pt-BR.lua.
- Simplified Chinese aliases such as zh-Hans.lua, zh-CN.lua, zh-CHS.lua, zh-Hans-CN.lua, zh-Hans-SG.lua, and zh-SG.lua load zh.lua.

Some translations were prepared with automated assistance and may contain errors or unnatural wording. I apologize for anything I got wrong.

Native speakers are encouraged to report or submit corrections through the SN2 POI Map GitHub project or on the Nexus mod page's bug reporting feature:
https://github.com/SPaladinoMods/SN2-POI-Map
